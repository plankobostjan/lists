# Extensions I use

This is a list of extensions I use in Firefox. Most of them are privacy and DF (Distracion Free) centered.

## Privacy
- [Decentraleyes](https://addons.mozilla.org/en-US/firefox/addon/decentraleyes/)
- [HTTPS Everywhere](https://addons.mozilla.org/en-US/firefox/addon/https-everywhere/)
- [Privacy Badger](https://addons.mozilla.org/en-US/firefox/addon/privacy-badger17/)
- [uBlock Origin](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)
- [Cookie AutoDelete](https://addons.mozilla.org/en-US/firefox/addon/cookie-autodelete/)

## DF
- [DF YouTube (Distraction Free)](https://addons.mozilla.org/en-US/firefox/addon/df-youtube/)
- [Replace YouTube's Home with Subscriptions](https://addons.mozilla.org/en-US/firefox/addon/ytmysubs/)

## Other
- [Tridactyl](https://addons.mozilla.org/en-US/firefox/addon/tridactyl-vim/)
- [Custom New Tab Page](https://addons.mozilla.org/en-US/firefox/addon/custom-new-tab-page/)
- [LastPass Password Manager](https://addons.mozilla.org/en-US/firefox/addon/lastpass-password-manager/)
- [Pinboard WebExtension](https://addons.mozilla.org/en-US/firefox/addon/pinboard-webextension/)
